Conducted an econometric analysis of Airbnb listing prices across London using Stata 18, achieving a HIGH DISTINCTION-LEVEL GRADE.
Built a hedonic pricing model with log-transformed price as the dependent variable, incorporating property characteristics, host attributes, and location controls (Central vs non-Central London).
Key methods:

OLS regression with robust standard errors
Interaction terms to test for differential effects across sub-regions
Heteroskedasticity diagnostics (e.g. Breusch-Pagan/White tests)
Discussion of potential endogeneity and omitted variable bias
Descriptive and correlation analysis of listing characteristics and price
